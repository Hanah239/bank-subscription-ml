# Bank Client Subscription Prediction

## Overview

This project focuses on predicting whether a bank client will subscribe to a term deposit using machine learning techniques. It is based on a real-world marketing dataset and aims to support data-driven decision-making.

## Tools & Technologies

* Python
* Pandas, NumPy
* Scikit-learn

## What I Did

* Cleaned and prepared the dataset, including encoding categorical variables
* Split data into training and testing sets (80/20)
* Trained and compared K-Nearest Neighbour and Decision Tree models
* Evaluated performance using accuracy, precision, recall, and F1-score

## Results

* Achieved up to 87% accuracy
* Identified trade-offs between accuracy and recall
* Selected model based on business relevance rather than accuracy alone

## Key Learning

This project highlighted the importance of choosing evaluation metrics carefully. In real-world scenarios, recall can be more important than accuracy when identifying potential customers.
